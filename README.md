## Установка

Приложение использует версию node.js 17.2.0.
```
 `git clone https://github.com/chOlya/website-ts.git`
```
Затем перейти в скаченный проект.
```
`npm install`
```
Для запуска введите 
```
`npm start`
```
Приложение будет запущено.
